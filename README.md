### 🤖 AI Agent Pipeline
Input Report
↓
🔍 Agent 1: GATEKEEPER
→ Scans input
→ Detects urgency (1-10)
→ Classifies domain
↓
🧠 Agent 2: STRATEGIC ANALYST
→ Fetches live Pakistan news
→ Calculates PKR impact
→ Selects action strategy
↓
⚡ Agent 3: EXECUTOR
→ Takes automated action
→ Sends Gmail alerts
→ Creates social posts
→ Shows Before/After results
