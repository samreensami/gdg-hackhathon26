# InsightFlow AI - Agent Rules

## Always follow:
- Pakistani context: use PKR, Pakistani cities, local companies
- Show BEFORE and AFTER state for every action
- Never use real personal data (use fictional company names)
- All logs must have timestamps (HH:MM:SS format)
- Financial figures in PKR with proper formatting
- Print emoji logs for visual appeal in terminal
- Every agent must return valid JSON
- Email drafts saved to output/email_drafts/
- Social posts saved to output/social_posts/

## Code standards:
- Clean modular Python code
- Comments on every function
- Try-except on every external call
- Rich library for terminal formatting
